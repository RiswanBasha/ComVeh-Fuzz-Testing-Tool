This project deals with the automation of the CAN bus Tx and Rx id's. In addition to that, This application will generate the CAPL script for the selected is's and selected anamolies for the easy interaction for testing. This is a BEG GMBH project.

Framework--> PyQt5
Languages--> Python,CAPL,Embedded C
Modules----> xlrd,pyqt5 libraries,Os,Sys,win32com.client
Tools------> Vector CANoe, Labcar, VSCode,QT Designer
